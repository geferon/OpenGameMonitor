Variables allowed in server parameters:

  • GameID = Game ID
  • GameName = Game Name
  • GameSteamID = Game SteamID
  • ServerID = Server's ID
  • ServerName = Server's Name
  • ServerIP = Server's IP
  • ServerPort = Server's Port
